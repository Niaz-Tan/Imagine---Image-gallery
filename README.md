# 📸 Image Gallery App

A full-stack image gallery web application built with Next.js using API routes and client-side rendering. The project features a modern UI, responsive masonry layout, and dynamic modal navigation using intercepting routes.

It includes full CRUD functionality handled through Next.js API routes with in-memory or JSON-based data management.

---

## 🚀 Features

* 🧱 Responsive masonry layout (done)
* 🪟 Modal image preview using intercepting routes (coming soon)
* 🔄 Full CRUD operations via API routes (coming soon)
* 🔗 Dynamic routing (modal + full page view) (done)
* ❤️ Likes, views, and share tracking (coming soon)
* 👤 Author information system (coming soon)
* 🏷️ Tag-based image organization (coming soon)

---

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Frontend:** React, Tailwind CSS
* **Backend:** Next.js API Routes
* **Data Source:** In-memory / JSON file

---

---

## 🧠 Key Concepts Used

* Next.js App Router architecture
* Intercepting routes for modal UX
* Dynamic routing system
* Client + API separation
* CRUD operations using API routes
* JSON / in-memory data handling
